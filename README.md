rapid-JS-prototyping-with-Parse
===============================

Rapid prototyping with Backbone front-end and Parse back-end.

Launched with the [Super Simple Backbone Starter Kit](https://github.com/azat-co/super-simple-backbone-starter-kit), which is a  This app has the following dependencies: RequireJS 2.0.2, RequireJS Text plug-in 2.0.0, jQuery 1.10.2, Backbone 1.10.2, Underscore 1.5.2, and Twitter Bootstrap 3.0.2. Require and Boostrap are locally hosted filed, while the others are hot-linked from CDNs.
